# java-microservices

java-microservices is a demo to show how micro services work. For this demo to run you will need:

# Prerequisites!
  - Java 8
  - Maven 3.6.3 (Binary Zip for Windows)
  - Spring Boot 2.2.5
  - MySql 8.0.19
  - Eclipse
  - Postman
